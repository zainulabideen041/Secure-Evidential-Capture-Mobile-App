## üîê Security & Confidentiality

### Special Obligations

Given the sensitive nature of this government application:

1. **Confidentiality**
   - All project information is confidential
   - Do not discuss project details publicly
   - Follow classification guidelines
   - Protect sensitive data at all times

2. **Security**
   - Report security issues immediately
   - Never commit sensitive data
   - Follow security best practices
   - Maintain secure development environments

3. **Compliance**
   - Adhere to government regulations
   - Follow data protection laws
   - Comply with evidence handling procedures
   - Respect chain of custody requirements

## üëÆ Enforcement

### Reporting Issues

If you experience or witness unacceptable behavior:

1. **Internal Reporting**
   - Report to your direct supervisor
   - Contact project lead: [email]
   - Use anonymous reporting system (if available)

2. **HR Department**
   - For serious violations
   - For harassment or discrimination
   - For workplace safety concerns

3. **Security Team**
   - For security violations
   - For confidentiality breaches
   - For compliance issues

### Confidentiality

All reports will be handled with discretion and confidentiality to the extent possible while ensuring proper investigation.

### Investigation Process

1. **Receipt**: Report acknowledged within 24 hours
2. **Review**: Initial assessment within 48 hours
3. **Investigation**: Thorough investigation conducted
4. **Resolution**: Appropriate action taken
5. **Follow-up**: Reporter notified of outcome
